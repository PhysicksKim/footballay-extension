# Footballay Extension

Chrome/Edge extension MVP for showing compact Footballay live match stats over football streaming pages.

## Stack

- WXT
- React
- TypeScript
- Manifest V3

## Commands

```bash
npm install
npm run dev
npm run build
npm run build:edge
npm run typecheck
```

Development starts with mock live-match data. Set `VITE_FOOTBALLAY_API_BASE_URL` when the real API endpoint is ready.
